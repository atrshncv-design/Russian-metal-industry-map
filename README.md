# Russian Metal Industry Map

Интерактивная карта металлургической промышленности России.

## Каноничная версия в этом репозитории

Этот репозиторий намеренно закрепляет **конкретную** версию карты из файла:

- `russia-metallurgy-map (6).html`

Опубликованная версия приложения находится в:

- `index.html`

При `git clone` или скачивании ZIP из этого репозитория должна загружаться именно эта версия карты.

## Проверка версии

Текущий `index.html`:

- размер: `64946` байт
- SHA-256: `cbefc6fe88f6f93e91799556d5a1568c5bfa1f7fc84d577ed576876f389d627c`

Подробности зафиксированы в [VERSION.md](./VERSION.md).

## Локальный запуск

Откройте файл `index.html` в браузере.

## Политика обновлений

Если в будущем загружается новая версия карты, обновление должно включать:

1. замену `index.html`
2. обновление `VERSION.md`
3. отдельный коммит с описанием источника новой версии
